=== TITLE ===

How to update the Heroku webserver's code to match changes to the source.

=== PROCEDURE: SETUP ===

1. heroku git:remote -a [revistatejos,textilesjournal]; to connect the heroku app to the master branch source code of this git project.

2. git remote rename heroku heroku-[revistatejos,textilesjournal]; to rename the origin from heroku to an app-specific name.

=== PROCEDURE: DEVELOP ===

3. git commit; any pending changes in the project.

4. git push; to origin/master.

5. git push heroku-[revistatejos,textilesjournal] master; to update heroku with the code on the master branch.

=== NOTES ===

(a) [apple,banana] refers to a choice of either apple or banana.
